# **App Name**: ConnectSphere

## Core Features:

- One-to-One Chat: Enable real-time messaging between users.
- Group Chat: Create and participate in group conversations.
- Ephemeral Status: Share temporary status updates that disappear after 24 hours, stored in Firebase Cloud Storage.
- User Profiles: Create and manage user profiles with personal information.
- Affiliate Product Showcase: Admin-managed affiliate products with links, displayed within the app.
- Admin Panel: Manage users, content, FCM, and app settings through a dedicated admin interface.
- Location-Based Suggestions: Suggest new chats based on user proximity. The tool will use the stored one-time location information collected during sign-up to prioritize the list of profiles, ensuring the nearest individuals appear at the top to facilitate connection.
- Content Sharing: Users can send multimedia files: images, texts, videos, files, and links.

## Style Guidelines:

- Primary color: Soft blue (#77B5FE), offering a sense of calmness and connectivity.
- Background color: Light gray (#F5F5F5), providing a clean and modern backdrop.
- Accent color: Warm orange (#FFB347), used for calls to action to grab user attention.
- Body font: 'PT Sans', sans-serif for a clear and friendly user interface.
- Headline font: 'Space Grotesk', sans-serif for a computerized, techy feel.
- Use a set of minimalistic line icons to ensure clarity.
- Dark mode: a theme should be avalable to the user from the setting panel
- Subtle animations for smooth transitions and feedback, such as loading indicators and message send confirmations.
